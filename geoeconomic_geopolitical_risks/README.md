# Geoeconomic and Geopolitical Regional Risks Dataset

## Overview

This folder contains the data pipeline used to construct a **regional geoeconomic and geopolitical risk dataset for Europe**. The dataset combines spatial geometries for **NUTS-2 regions** with regional economic and demographic indicators derived from Eurostat.

The objective is to provide a reproducible dataset that can be used in the context of **geospatial data science**, spatial economic analysis, and regional political economy research. The dataset allows the exploration of spatial patterns of economic development, labour market conditions, industrial restructuring, and demographic dynamics across European regions.

This material accompanies the broader **Geospatial Data Science** project and illustrates how official statistical sources can be combined with spatial data to produce research-ready datasets.



## Spatial Units

The dataset uses **NUTS-2 regions**, the official territorial classification used by the European Union for regional statistics and regional policy analysis.

NUTS-2 regions represent an intermediate spatial scale that is widely used for studying regional economic development and territorial disparities.



## Variables

The dataset includes the following regional indicators.

| Variable               | Description                                                      |
| ---------------------- | ---------------------------------------------------------------- |
| `nuts_id`              | NUTS-2 regional identifier                                       |
| `region_name`          | Name of the region                                               |
| `country`              | Country code                                                     |
| `gdp_pps_eu27`         | GDP per capita in purchasing power standards (EU27 = 100)        |
| `unemployment_rate`    | Regional unemployment rate                                       |
| `net_migration_rate`   | Net migration rate                                               |
| `manufacturing_emp`    | Employment in manufacturing                                      |
| `total_emp`            | Total employment                                                 |
| `manufacturing_share`  | Share of manufacturing employment                                |
| `manufacturing_change` | Change in manufacturing employment share relative to a base year |

These indicators capture several key dimensions of regional economic structure:

* regional prosperity,
* labour market conditions,
* industrial specialization,
* demographic dynamics.



## Data Sources

All indicators are derived from official statistical sources.

| Source                     | Description                                  |
| -------------------------- | -------------------------------------------- |
| Eurostat                   | Regional economic and demographic indicators |
| GISCO                      | Spatial geometries for NUTS regions          |
| Eurostat regional accounts | Employment by economic sector                |

The main Eurostat datasets used include:

* `tgs00006` – GDP per capita in purchasing power standards
* `tgs00010` – Regional unemployment rates
* `tgs00099` – Net migration rates
* `nama_10r_3empers` – Regional employment by economic activity

## Reproducibility

The dataset is generated entirely through R code using publicly available statistical sources. The script included in this repository automatically downloads the required Eurostat datasets and reconstructs the dataset.

To regenerate the dataset locally, run:

```r id="tcrmn8"
source("geoeconomic_geopolitical_risks/code/build_europe_dataset.R")
```



## Intended Use

This dataset is designed for teaching and research in:

* geospatial data science,
* regional economic analysis,
* spatial visualization,
* exploratory spatial analysis,
* introductory spatial econometrics.

It provides a reproducible example of how spatial datasets can be constructed from official statistical sources.
